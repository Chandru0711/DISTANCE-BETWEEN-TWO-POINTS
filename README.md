## AIM:

## ALGORITHM:

### Step 2:

### Step 3:

 d=√((x_2-x_1)²+(y_2-y_1)²)
### Step 4: 
Print the values

### Step 5: 
End the program

### PROGRAM:
```python
#Program to find the distance between two points.
#Developed by:CHANDRU SM

import math
l1=[4,2]
l2=[10,6]
distance=math.sqrt(((l2[0]-l1[0])**2+((l2[1]-l1[1])**2)))
print("{:.2f}".format(distance))
```
